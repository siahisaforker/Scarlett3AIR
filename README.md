# Ongoing source port of Sonic 3 A.I.R. using ANGLE and SDL2

Source to build the Oxygen Engine and Sonic 3 A.I.R. for Xbox consoles in developer mode


# How is this different from the existing port?

I used the existing port. It's a web app basically and incredibly unintuitive, while also being unmaintained. This port will stay up-to-date with the main branch of Sonic 3 A.I.R. unless said otherwise.
